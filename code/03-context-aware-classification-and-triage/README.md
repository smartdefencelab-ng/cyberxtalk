# Code Track 3: Classification and Triage Implementation

This folder holds implementation artifacts for the classification and triage paper.

## Intended Artifacts

- baseline rule-based classifier
- TF-IDF / classical ML experiments
- transformer experiments
- severity scoring logic
- triage pipeline
- evaluation scripts

## Immediate Goal

Build a simple classification and severity-scoring pipeline that converts informal reports into incident category, priority, and recommended response actions.
