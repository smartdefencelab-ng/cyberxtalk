# Code Track 4: Multilingual Low-Resource NLP Implementation

This folder holds implementation artifacts for the multilingual low-resource NLP paper.

## Intended Artifacts

- language detection experiments
- normalization scripts
- code-mixed incident report samples
- STT baseline tests
- YarnGPT TTS / ASR experiments
- preprocessing and evaluation scripts

## Immediate Goal

Create a practical language-normalization and voice-processing layer that supports CyberXTalk incident intake without overclaiming language coverage.
